# 2026年超予測会

- [connpassイベントページ](https://fin-py.connpass.com/event/381982/)

## ダッシュボード

**[スコア分析ダッシュボード](https://fin-py.github.io/super-prediction-2026/)**

17件の予測データを可視化したインタラクティブダッシュボード（Stlite + Plotly）

### 含まれるグラフ

1. 予測別スコア内訳（積み上げ棒グラフ）
2. スコア分布（ストリッププロット）
3. 箱ひげ図
4. 散布図 - 実現可能性 vs 独自性の負の相関
5. ダンベルチャート - スコア間のギャップ可視化
6. 四象限チャート - 予測の分類
7. データ一覧
